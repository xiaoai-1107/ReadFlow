# 第一阶段测试计划与验收记录

## 测试目标

验证第一阶段软件链路是否真实成立，而不是停留在静态展示层。

重点关注：

- 导入是否真实入库
- 解析结果是否真实持久化
- 原文/译文模式是否真实联通
- 高亮/tag 是否真实写回
- 阅读恢复是否真实命中会话
- 已缓存内容是否具备离线可读基础
- PC 与移动端是否共享同一套交互语义
